# ⚠️ СРОЧНОЕ ОБНОВЛЕНИЕ СЕРВЕРА

## Проблема:
- В Git есть новые изменения (6 минут назад)
- На сервере старая версия (4 часа назад)
- Изменения не применяются автоматически!

## Решение - выполните на сервере:

```bash
# 1. Перейдите в корневую директорию проекта
cd /ssd/www/bigo-mvp

# 2. Проверьте текущую версию
git log --oneline -1

# 3. Получите последние изменения из Git
git pull origin master

# 4. Проверьте, что изменения получены
git log --oneline -1
# Должен показать: "Добавлен JavaScript код для удаления overlay в StreamBroadcaster"

# 5. Перейдите в директорию фронтенда
cd frontend/web

# 6. Остановите PM2
pm2 stop nio-frontend

# 7. Удалите старую сборку (ВАЖНО!)
rm -rf .next

# 8. Пересоберите проект
npm run build

# 9. Запустите PM2 снова
pm2 restart nio-frontend

# 10. Проверьте статус
pm2 status

# 11. Проверьте логи (не должно быть ошибок)
pm2 logs nio-frontend --lines 30
```

## Если git pull не работает (конфликты):

```bash
cd /ssd/www/bigo-mvp

# Отмените все локальные изменения на сервере
git checkout -- .

# Получите изменения
git pull origin master

# Затем выполните шаги 5-11 выше
```

## Проверка после обновления:

1. Откройте сайт: https://bigo.1tlt.ru
2. Откройте страницу стрима
3. Overlay с ID и логотипом должен исчезнуть
4. Проверьте в консоли браузера (F12) - не должно быть ошибок

## Если overlay все еще виден:

1. Очистите кэш браузера (Ctrl+Shift+Delete)
2. Обновите страницу с принудительной перезагрузкой (Ctrl+F5)
3. Проверьте логи PM2: `pm2 logs nio-frontend --lines 50`
