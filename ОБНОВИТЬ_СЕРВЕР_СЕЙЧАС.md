# ⚠️ ОБНОВИТЬ СЕРВЕР СЕЙЧАС

## Проблема:
На сервере последний коммит: "Улучшена генерация credentials..." (старый)
В Git есть новые коммиты с удалением overlay (новые)

## Решение:

```bash
# 1. Перейдите в КОРНЕВУЮ директорию проекта (ВАЖНО!)
cd /ssd/www/bigo-mvp

# 2. Проверьте текущую версию
git log --oneline -1

# 3. Получите последние изменения из Git
git pull origin master

# 4. Проверьте, что изменения получены
git log --oneline -1
# Должен показать: "Добавлен JavaScript код для удаления overlay в StreamBroadcaster"

# 5. Перейдите в директорию фронтенда
cd frontend/web

# 6. Остановите PM2
pm2 stop nio-frontend

# 7. Удалите старую сборку (ОБЯЗАТЕЛЬНО!)
rm -rf .next

# 8. Пересоберите проект
npm run build

# 9. Запустите PM2 снова
pm2 restart nio-frontend

# 10. Проверьте статус
pm2 status

# 11. Проверьте логи
pm2 logs nio-frontend --lines 30
```

## Если git pull показывает конфликты:

```bash
cd /ssd/www/bigo-mvp

# Отмените все локальные изменения
git checkout -- .

# Получите изменения
git pull origin master

# Затем выполните шаги 5-11 выше
```

## Проверка:

После выполнения команд проверьте:
1. `git log --oneline -1` - должен показать новый коммит
2. Сайт должен работать без overlay с ID
